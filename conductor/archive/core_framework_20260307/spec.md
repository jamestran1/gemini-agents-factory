# Track: Implement Core 5-Role Collaborative Agent Framework

## Goal
Establish the foundational collaborative chat environment allowing 5 distinct Gemini CLI subagents (Product Owner, Designer, Project Manager, Software Engineer, QA) to operate sequentially and collaboratively to fulfill a software project request.

## Requirements
- Define agent roles, system prompts, and tool access boundaries.
- Implement the "Central Orchestrator" responsible for delegating tasks.
- Provide a shared context (message bus) so agents can converse and debate requirements.
- Integrate initial Trello scaffolding to manage tasks generated by the Project Manager.
